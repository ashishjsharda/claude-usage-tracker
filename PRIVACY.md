# Privacy Policy for Claude Usage Tracker

**Last Updated:** October 31, 2025

## Overview

Claude Usage Tracker is committed to protecting your privacy. This privacy policy explains how our Chrome extension handles data.

## What Data We Collect

Claude Usage Tracker collects and stores the following data **locally on your device**:

- **Message Count:** Number of messages sent on claude.ai
- **Chat Sessions:** Number of conversations started
- **Usage Timestamps:** When you use Claude AI
- **Model Type:** Whether you're using Claude Sonnet or Opus
- **Activity Data:** Weekly usage patterns

## What We DON'T Collect

We explicitly do NOT collect, access, or store:

- ❌ Message content or conversation text
- ❌ Personal information
- ❌ Claude API keys
- ❌ Login credentials
- ❌ Browsing history outside claude.ai
- ❌ Any identifiable information

## How Data is Stored

All data is stored **locally in your browser** using Chrome's LocalStorage API. This means:

- ✅ Data never leaves your device
- ✅ No external servers or databases
- ✅ No cloud storage
- ✅ Complete local control

## How Data is Used

The collected data is used solely to:

1. Display your usage statistics in the extension popup
2. Generate activity charts and visualizations
3. Calculate your usage streak
4. Show weekly usage patterns
5. Track model preference (Sonnet vs Opus)

## Data Sharing

We do NOT:

- Share data with third parties
- Sell data to anyone
- Send data to external servers
- Use analytics or tracking services
- Transmit data outside your browser

**Your data stays with you, always.**

## Third-Party Services

This extension does not integrate with any third-party services, analytics platforms, or external APIs.

## User Rights and Control

You have complete control over your data:

### View Your Data
- All data is visible in the extension popup
- No hidden or background collection

### Export Your Data
- Click "Export Data" to download your usage as JSON
- Full transparency into what's collected

### Delete Your Data
- Click "Reset Stats" to clear all data
- Uninstalling the extension removes all data
- No residual data remains

## Permissions Explained

The extension requests the following permissions:

### Storage Permission
**Why:** To save your usage statistics locally
**Data Stored:** Message counts, timestamps, streak data
**Location:** Your browser's LocalStorage (local only)

### Tabs Permission
**Why:** To detect when you navigate to claude.ai
**Usage:** Only monitors claude.ai URL changes
**Limitation:** Cannot access tab content

### Host Permission (https://claude.ai/*)
**Why:** To monitor activity on Claude AI website
**Usage:** Detects new messages and chat sessions
**Limitation:** Only active on claude.ai, nowhere else

## Children's Privacy

This extension does not knowingly collect information from children under 13. The extension is designed for users of Claude AI, which requires users to be 18 or older.

## Data Security

We prioritize security:

- All data stored using Chrome's secure LocalStorage
- No network transmission of personal data
- No vulnerabilities from external dependencies
- Open source code for transparency
