# Quick Start Guide

Get up and running with Claude Usage Tracker in under 2 minutes! ‚ö°

## Installation

### Option 1: From GitHub (Available Now)

1. **Download the extension**
   ```bash
   git clone https://github.com/ashishjsharda/claude-usage-tracker.git
   ```
   
   Or download as ZIP from GitHub and extract

2. **Open Chrome Extensions**
   - Type `chrome://extensions/` in your address bar
   - Press Enter

3. **Enable Developer Mode**
   - Toggle the switch in the top-right corner

4. **Load the Extension**
   - Click "Load unpacked"
   - Select the `claude-usage-tracker` folder
   - Extension icon should appear in your toolbar!

### Option 2: Chrome Web Store (Coming Soon)

1. Visit the [Chrome Web Store page](#)
2. Click "Add to Chrome"
3. You're done! üéâ

## First Use

1. **Visit Claude.ai**
   - Go to [claude.ai](https://claude.ai)
   - Sign in to your account

2. **Start Using Claude**
   - Have a conversation with Claude
   - Send a few messages

3. **Check Your Stats**
   - Click the extension icon in your toolbar
   - See your usage statistics update in real-time!

## Troubleshooting

### Extension Not Tracking?

1. **Refresh the Page**
   - Press F5 or Ctrl/Cmd + R on claude.ai
   
2. **Reload the Extension**
   - Go to `chrome://extensions/`
   - Click the refresh icon on Claude Usage Tracker
   
3. **Check Permissions**
   - Extension needs access to claude.ai
   - Make sure it's enabled

### Stats Not Showing?

1. **Send a Test Message**
   - Start a new conversation
   - Send a message
   - Click extension icon
   
2. **Clear Storage & Reinstall**
   - Right-click extension icon
   - Click "Options" (if available)
   - Or reinstall the extension

### Model Detection Wrong?

- Model detection is best-effort
- Based on UI indicators
- May need page refresh to detect
